**Tugas Proyek: Langkah Implementasi**
===================================================

**1. Buat Struktur Direktori**
--------------------------------------------------
Gunakan mkdir untuk membuat folder: documents/, images/, archives/, logs/. Buat 20+ file sample dengan touch dan echo.

**Hasil Pengerjaan --> BuatStrukturDirektori.sh**

**2. Script Organisasi File**
--------------------------------------------------
Tulis script yang menggunakan find, mv, dan cp untuk memindahkan file ke folder yang sesuai berdasarkan ekstensi.

**Hasil Pengerjaan --> ScriptOrganisasiFile.sh**

**3. Fungsi Pencarian**
--------------------------------------------------
Implementasikan pencarian file menggunakan find dan grep. Buat fungsi untuk mencari berdasarkan nama, ukuran, dan konten.

**Hasil Pengerjaan --> FungsiPencarian.sh**

**4. Generate Laporan**

--------------------------------------------------
Gunakan ls, wc, du, dan piping untuk membuat laporan statistik file sistem. Output ke file report.tx

**Hasil Pengerjaan --> reportsyntax.sh --> report.txt**
